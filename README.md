## 🎵 About

熵尾音 クロ (Shang Wei Yin Kuro) is a UTAU voicebank featuring a gentle wolf furry character.

- **Type**: VCV Voicebank
- **Scale**: B3 (Single Scale)
- **Recommended Range**: G2~C5
- **Recommended Engine**: Worldline-R
- **Voice Character**: Confident, Innocent
