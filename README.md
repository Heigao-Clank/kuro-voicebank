## 🎙️ 音源规格 (Voicebank Specs)

基于原始录音制作的日语连续音（VCV）声库，推荐使用 [OpenUTAU](https://github.com/stakira/OpenUtau) 配合 Worldline-R 引擎。

| 项目 | 规格 |
| :--- | :--- |
| 收录方式 | 日语连续音 `JA VCV` |
| 收录音阶 | B3（单音阶） |
| 推荐音域 | G2 ~ C5 |
| 推荐引擎 | `Worldline-R` |
| 声线特征 | 自信、懵懂 |

### 💡 快速上手

1. 将下载的声库压缩包拖入 [OpenUTAU](https://github.com/stakira/OpenUtau) 窗口
2. 文本编码选 `utf-8`，歌手类型选 `utau`，按提示完成安装
3. 音轨中选择 `熵尾音 クロ`，引擎切换为 `Worldline-R` 即可开始编辑

## ⚖️ 使用协议 (Usage Guidelines)

本摘要仅供参考，完整条款请阅读 [官方网站协议页面](https://kuro-voicebank.pages.dev/#guidelines)。**使用本音源即表示您同意全部条款。**

<details>
<summary><b>✅ 允许的事项（无需授权）</b></summary>

* **创作与发布** — 个人非商业使用、视频投稿、直播、同人范围内收益化（含周边贩售）
* **二次创作** — 翻唱、Remix、作为通用语音素材（无需角色名义）
* **AI 后期** — 允许 AI 工具修音/混音；允许**本地私有**模型训练（仅供个人辅助调声）
* **文件修改** — 允许修改 `oto.ini` 及 `.frq` 并分享
* **署名** — 非强制，标注的话非常感谢

</details>

<details>
<summary><b>❌ 严格禁止的事项</b></summary>

* 分发原始音频样本（`.wav`）或将音源打包进收费产品
* 伪造制作者信息
* 用于违法、仇恨、歧视或损害角色形象的内容
* **AI 红线：** 将训练模型（二次声库）开源/分享/售卖；将音源作为公共 AI 模型数据集上传；用于商业 AI 开发

</details>

<details>
<summary><b>⚠️ 需联系作者授权的事项</b></summary>

* 正式商业使用（企业 PR、广告、商业平台收录等）
* 虚拟主播（Vtuber）皮套使用
* 游戏内置语音
* 其他不确定的情况
