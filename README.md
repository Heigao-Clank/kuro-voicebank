## 🎤 关于音源

熵尾音 クロ（Shang Wei Yin Kuro）是一个 VCV 日语音源，双尾狼兽人角色。

**声线特点**：自信、懵懂

| 规格 | 参数 |
|------|------|
| 类型 | JA VCV |
| 音阶 | B3 单音阶 |
| 推荐音域 | G2 ~ C5 |
| 推荐引擎 | Worldline-R |
