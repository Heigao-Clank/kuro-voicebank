## 🎙️ 音源规格 (Voicebank Specs)

本音源为基于原始录音制作的日语连续音（VCV）声库，推荐使用 [OpenUTAU](https://github.com/stakira/OpenUtau) 配合 Worldline 引擎以获得最佳效果。

* **收录语言：** 日语连续音 (`JA VCV`)
* **收录音阶：** B3（单音阶）
* **推荐音域：** G2 ~ C5
* **推荐引擎：** `Worldline-R`
* **声线特征：** 自信、懵懂

### 💡 快速上手指南
如果您使用的是 OpenUTAU：
1. 将下载的声库压缩包拖入 OpenUTAU 窗口。
2. 选择文本编码为 `utf-8`，歌手类型为utau，按指示完成安装。
3. 在音轨上选择 `熵尾音 クロ`。
4. 将引擎 (Resampler) 切换为 `Worldline-R` 即可开始编辑。
